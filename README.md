# Regional Mobility Visor

An interactive dashboard for exploring mobility scenarios in Gipuzkoa. The application allows users to modify mobility-related variables and visualize their impact on a set of key regional mobility indicators in real time.

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/City-Science-Lab-Gipuzkoa/regional_mobility.git
cd <repository-folder>
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the application

```bash
python app_distributed.py
```

A mapbox API key is required to execute the adequate visualization.
